### 技術問題

- 後端: 參考 README.md API 文檔

### 常見問題

**Q: 為什麼模型訓練沒有反應?**  
A: 確認後端服務器已啟動。檢查 WebSocket 連接狀態。

**Q: 如何自訂演算法選擇?**  
A: 在 `js/app.js` 中修改 `algorithm-grid` 的 HTML 標記。

**Q: 如何改變配色?**  
A: 編輯 `css/style.css` 中的 `:root` CSS 變數。
