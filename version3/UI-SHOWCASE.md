#### 滑塊設計
```
[■■■■□] ← 漸層色滑塊
 ^ 拇指（白色 + 藍色邊框）
```

#### 配色
- **滑塊背景**：紅 → 橙 → 綠漸層
  - 0-33%：紅色 (#ef4444) - 低效
  - 33-66%：橙色 (#f59e0b) - 中等
  - 66-100%：綠色 (#10b981) - 高效
- **拇指**：白色 + 2px 藍色邊框
- **結果背景**：淡灰 (#f1f5f9)

## 🎨 全局色彩系統

### 主色板
| 用途 | 顏色 | Hex | 使用 |
|------|------|-----|------|
| 主品牌 | 科技藍 | #1e40af | 按鈕、邊框、圖標 |
| 淺藍 | 淡藍 | #3b82f6 | 背景、陰影、次要 |
| 成功 | 綠色 | #10b981 | ✓ 成功、正向、上線 |
| 警告 | 橙色 | #f59e0b | ⚠️ 警告、中立、測試 |
| 錯誤 | 紅色 | #ef4444 | ❌ 錯誤、下架、否定 |
| 背景 | 淡灰 | #f8fafc | 主背景 |

### 應用規則
- **信任與穩定感**：主要用藍色
- **正向反饋**：必用綠色
- **風險警示**：必用紅色/橙色
- **對比度**：確保 WCAG AA 標準以上

## 🎯 使用者體驗設計亮點

### 1. **視覺層次**
- 標題：28px、黑色、粗體
- 副標題：14px、灰色
- 身體文本：14px、黑色
- 輔助文本：12px、淡灰色

### 2. **可訪問性**
- 色彩對比度 ≥ 4.5:1
- 字體大小最小 12px
- 滑塊拇指尺寸 ≥ 20px
- 按鈕最小點擊區域 44×44px

### 3. **一致性**
- 所有圓角：6px (sm) / 8px (md) / 12px (lg)
- 所有間距：5px 倍數（5, 10, 15, 20, 25, 30...）
- 所有陰影：統一 3 個層級
