|------|--------|------|

| 實時結果更新 | JavaScript 事件監聽 | app.js L222-243 |
| 動態決策建議 | 生成邏輯 | app.js L244-260 |
| 百分比顯示 | 預測購買機率 (%) | index.html L456-461 |

**What-If 預測模型（簡化算法）：**
```javascript
預測 = 基線(50%) +
       行銷預算* 0.25 -
       交貨延遲 * 0.1 +
       滿意度 * 0.2 -
       價格差異 * 0.15
```
